# Monad Jumper

A blockchain-based jumping game built on Monad testnet.

## Features
- Jump mechanics with blockchain transactions
- Score tracking on Monad testnet
- Beautiful UI/UX design
- Responsive gameplay
- Transaction bundling for efficiency

## Tech Stack
- React
- Vite
- Ethers.js
- Monad Testnet
- HTML5 Canvas
- Supabase
